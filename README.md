### 运行环境

- Node.js 16.14+

- pnpm 8.x+

- Python3.8-3.11

screenshot
![image](https://github.com/tansen87/JET-template/assets/98570790/43161ddb-3afd-4b96-9251-98d608964c60)


### 初始化

```shell
pnpm run init
```

### dev

```shell
pnpm run start
```

### build

```shell
# no console window
pnpm run build

# with console window
pnpm run pre
```
